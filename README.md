# SaaS Лендинг с Шейдерами

![Preview](https://cdn.poehali.dev/templates/saas-landing-ru/preview-ru-v2.jpg)

Современный лендинг для SaaS-продуктов и digital-агентств с эффектным hero-экраном на WebGL шейдерах. Горизонтальная навигация между секциями, магнитные кнопки, кастомный курсор и grain-эффект создают премиальное ощущение. Включает секции: главная, портфолио, услуги, о компании и контакты.
